A lightweight, efficient browser logging library for tracking errors, console outputs, and network requests.

Browser TypeScript Logging

Installation

npm install loggify-lite

Features

  • Error tracking
  • Console logging (log, warn, error)
  • Network request monitoring (Fetch & XHR)
  • Configurable sampling rates
  • IndexedDB storage
  • Export capabilities

Quick Start

import { initializeLogger } from 'loggify-lite';

initializeLogger({
    maxDBSize: 5 * 1024 * 1024, // 5MB
    storeName: 'logs',
    dbName: 'loggify-db',
    maxEvents: 1000,
    windowMs: 60000,
    samplingRates: {
        ERROR: 1,
        CONSOLE_ERROR: 1,
        CONSOLE_LOG: 0.1,
        CONSOLE_WARN: 0.5,
        FETCH: 0.2
    },
    allowConsoleLogs: true,
    allowConsoleErrors: true,
    allowConsoleWarnings: true,
    allowNetworkRequests: true,
    allowErrors: true
});
