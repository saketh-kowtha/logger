Loggify-lite

A lightweight logging library for browser-based JavaScript applications.

Installation

npm install loggify-lite

Features

  • Console logging capture (errors, warnings, logs)
  • Network request tracking (Fetch API and XMLHttpRequest)
  • Unhandled error tracking
  • Export capabilities for logged events

Usage

import { initializeLogger, exportEvents } from 'loggify-lite';

// Initialize with your preferred configuration
initializeLogger({
    allowErrors: true,
    allowConsoleErrors: true,
    allowConsoleLogs: true,
    allowConsoleWarnings: true,
    allowNetworkRequests: true
});
